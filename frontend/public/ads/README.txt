Рекламные медиа для левой панели первого экрана (AdPanel).

Как добавить:
1) Положи сюда файлы — картинки (.jpg/.png/.webp) и/или видео (.mp4).
2) Открой frontend/src/components/AdPanel.jsx и перечисли их в AD_MEDIA, например:

   const AD_MEDIA = [
     { type: 'image', src: '/ads/ad1.jpg' },
     { type: 'image', src: '/ads/ad2.jpg' },
     { type: 'video', src: '/ads/promo.mp4' },
   ];

Пути начинаются с /ads/ (папка public раздаётся в корень сайта).

Поведение:
- картинка висит SLIDE_INTERVAL_MS (по умолчанию 6 сек), затем плавно сменяется;
- видео проигрывается целиком, потом переходит к следующему медиа;
- одно видео в списке — зациклится; одна картинка — будет статичной;
- пустой список — показывается плейсхолдер «Тут могла быть ваша реклама».

Важно: видео автоматически воспроизводится без звука (требование браузеров).
